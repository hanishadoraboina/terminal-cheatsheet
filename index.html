Quick Terminal Cheat Sheet

pwd

Description: Shows your current working directory.

Example: Use pwd to confirm which project folder you're in.

ls -la

Description: Lists all files, including hidden files.

Example: Run ls -la after cloning a Git repository.

cd

Description: Changes the current directory.

Example: Move into your project folder using cd portfolio.

mkdir

Description: Creates a new directory.

Example: Run mkdir portfolio to create a project folder.

touch

Description: Creates a new empty file.

Example: Run touch index.html to create a webpage.

cp

Description: Copies files or folders.

Example: Use cp report.txt backup.txt to make a backup.

mv

Description: Moves or renames files.

Example: Rename a file using mv old.txt new.txt.

rm

Description: Deletes files or folders.

Example: Delete an unwanted file using rm notes.txt.

grep

Description: Searches for matching text.

Example: Run ls | grep ".html" to find HTML files.

chmod +x

Description: Makes a script executable.

Example: Run chmod +x setup.sh before executing ./setup.sh.
