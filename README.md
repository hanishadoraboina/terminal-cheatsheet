## 📖 About the Project

**Terminal Cheat Sheet** is an interactive single-page web application that introduces beginners to the **10 most essential shell commands** used in everyday development.

Instead of presenting commands as plain text, the website provides an engaging terminal-inspired interface where each command includes:

- 📌 The command itself
- 📝 A simple one-line explanation
- 💡 A practical real-world example

This project was created as part of **Module 1 – Terminal Basics** to make learning the command line more approachable and visually engaging.

## 📸 Commands Included

| Command | Purpose |
|----------|----------|
| `pwd` | Display the current working directory |
| `ls -la` | List all files, including hidden files |
| `cd` | Navigate between directories |
| `mkdir` | Create a new directory |
| `touch` | Create a new empty file |
| `cp` | Copy files or directories |
| `mv` | Move or rename files |
| `rm` | Remove files or directories |
| `grep` | Search for matching text |
| `chmod +x` | Make a file executable |
